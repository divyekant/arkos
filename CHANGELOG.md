## [0.1.0] - 2026-03-01

### Added
- Astro + Starlight project scaffold
- Custom landing page with gold particle constellation, aurora glow, grain overlay
- Hero section with metallic shimmer text and dual CTAs
- Problem statement section with capability pills
- Tool showcase with 3x3 holographic card grid (rotating borders, cursor spotlight, tilt)
- Animated ecosystem diagram with flowing connections and hover highlighting
- Terminal typing demo with simulated carto index output
- Glassmorphism sticky navigation
- Footer with navigation links
- Documentation for all 9 ecosystem tools (Memories, Carto, Apollo, Delphi, Hermes, Conductor, SwarmEngine, PersonaSmith, Learning)
- Ecosystem architecture page
- About page with philosophy and contributing guide
- Gold Shiki syntax highlighting theme
- Astro view transitions for smooth navigation
- Dark theme with gold/amber accent palette
- Apollo project configuration
